# appdf-publisher

Publisher for [AppDF](https://github.com/onepf/AppDF).

## Usage

```shell
git clone git://github.com/rithis/appdf-publisher.git
cd appdf-publisher
./bin/appdf --username GOOGLE_PLAY_EMAIL --password GOOGLE_PLAY_PASSWORD PATH_TO_APPDF
```

## Testing

```shell
git clone git://github.com/rithis/appdf-publisher.git
cd appdf-publisher
make
```
